Dotfiles

Linux configuration files. Some of this is useful in OSX, however, I mostly use
different tools on that platform. So it is easier to pick the eyes out of what
I have here, than try to support both OSX and Linux simultaneous.

 * `gdbinit` GDB convenience macros.
 * `marc.bash` Bash customization.
 * `marc.el` Emacs customization.
 * `marc-gdb.py` GDB extensions for debugging C++.
